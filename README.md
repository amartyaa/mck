# mck - Multi-Cloud Kubernetes CLI

> One tool. Every cloud. All your clusters.

A unified CLI for managing Kubernetes clusters across AWS EKS, Azure AKS, GCP GKE, and Oracle OKE.

## Installation

```bash
go install github.com/amartyaa/mck@latest
```

## Quick Start

```bash
mck list
mck connect prod-eks-us-east
```
